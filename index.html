🚀 Run it yourself

Four steps. Five minutes. Zero cost.

1
Get a free Gemini API key
Go to aistudio.google.com → Get API Key. Free tier. No billing.
2
Clone and add your key
git clone https://github.com/vichitravird/otter-extension.git
echo 'const GEMINI_API_KEY = "your-key-here";' > config.js
3
Load in Chrome
Go to chrome://extensions → enable Developer mode → Load unpacked → select the folder.
4
Start writing
Click any text box on any website. The 🦦 button appears. Click it. Done.

Bug #1 — git wouldn't push
Tried to push before making the first commit. Git doesn't create a branch until there's something to commit.
broken
git push -u origin main
error: src refspec main
does not match any
fixed
git add .
git commit -m "..."
git push -u origin main
🐛
Bug #2 — sidebar couldn't read the text box
The sidebar ran as an iframe — sandboxed from content.js. chrome.runtime.sendMessage doesn't work across iframe boundaries.
broken
chrome.runtime
.sendMessage() ❌
fixed
window.parent
.postMessage() ✅
🐛
Bug #3 — GEMINI_API_KEY is not defined
sidebar.js loaded before config.js. The variable didn't exist yet when the script ran.
broken
<script src="sidebar.js">
<script src="config.js">
fixed
<script src="config.js">
<script src="sidebar.js">
🐛
Bug #4 — Gemini returned 404
The model name gemini-1.5-flash was outdated. Google updated the API endpoint.
broken
gemini-1.5-flash
404 Not Found ❌
fixed
gemini-2.0-flash
200 OK ✅
fix: update Gemini model to gemini-2.0-flash
🔐
Oops — config.js accidentally pushed to GitHub
The API key was committed and pushed. Rotated the key immediately, then scrubbed config.js from git tracking and history entirely using git filter-branch.

in GitHub history ❌
fixed

chore: remove config.js from tracking
v2 — UX improvements
📍
Update #1 — Grammarly-style inline anchor button
Replaced the fixed floating pill button with an inline anchor that attaches to the bottom-right corner of whatever text box you're focused on. Repositions on scroll and resize automatically.
feat: replace fixed button with Grammarly-style inline anchor
💬
Update #2 — replaced sidebar with compact popover
The full-height sidebar was covering the text box. Replaced it with a 320px popover card that floats above the anchor button. Removed sidebar.html, sidebar.js, sidebar.css entirely — all logic now lives in content.js.
feat: replace sidebar with compact inline popover card
🧹
Update #3 — cleaned up the AI response
Gemini was returning asterisks, "Here is your response" intros, and multiple options. Tightened the system prompt with strict rules: one message, no markdown, no preamble, nothing but the final text.
