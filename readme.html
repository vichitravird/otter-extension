🚀 Run it yourself

Four steps. Five minutes. Zero cost.

1
Get a free Gemini API key
Go to aistudio.google.com → Get API Key. Free tier. No billing.
2
Clone and add your key
git clone https://github.com/vichitravird/otter-extension.git
# create config.js (this file is gitignored)
echo 'const GEMINI_API_KEY = "your-key-here";' > config.js
3
Load in Chrome
Go to chrome://extensions → enable Developer mode → click Load unpacked → select the folder.
4
Start writing
Go anywhere. Click any text box. Hit the 🦦 button. Done.

📖 The build diary

The full story — wins, bugs, and everything in between.

📋

Phase 2 — built it
Five files built with deliberate Cursor AI prompts: manifest.json → content.js → sidebar.html → sidebar.js → icons. Each one reviewed and understood before committing.
feat: add sidebar.js with Gemini API integration
🐛
Bug #1 — git wouldn't push
Tried to push before making the first commit. Git doesn't create a branch until there's something to commit.
broken
git push -u origin main
error: src refspec main
does not match any
fixed
git add .
git commit -m "..."
git push -u origin main
🐛
Bug #2 — sidebar couldn't read the text box
The sidebar is an iframe. Iframes are sandboxed — chrome.runtime.sendMessage doesn't reach content.js from inside one.
broken
chrome.runtime
.sendMessage() ❌
fixed
window.parent
.postMessage() ✅
🐛
Bug #3 — GEMINI_API_KEY is not defined
sidebar.js loaded before config.js. The key variable didn't exist yet. Fixed by loading config.js first in sidebar.html.
broken
<script src="sidebar.js">
<script src="config.js">
fixed
<script src="config.js">
<script src="sidebar.js">
🐛
Bug #4 — Gemini returned 404
The model name gemini-1.5-flash was outdated. Google had updated the endpoint.
broken
gemini-1.5-flash
404 Not Found ❌
fixed
gemini-2.0-flash
200 OK ✅
fix: update Gemini model to gemini-2.0-flash
